 - 部分型

 - 変異アノテーション
   - ``extends List[+A]`` など。読んだけどよくわからなかった

 - 可変数引数
   - 引数宣言時に、 ``value: Type*`` で宣言可能。Seq[Type]で受け取れる

 - 部分適用とカリー化
   - 後々非常に重要な概念になりそう。ちょっとまだよくわかってない

 - 末尾再帰
   - これもScala の実行速度向上に必須のテクニックになりそう。まだよくわかってない。

 - 代数的データ型
   - １つ以上の以上のデータコンストラクタによって定義されるデータ型
     - データ型がそのデータコンストラクタの和であり
     - 各データコンストラクタがその引数の積
   - 何をいってるかよくわからない
